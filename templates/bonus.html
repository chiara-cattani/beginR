Tips & Tricks

R Productivity Tips
  • Use Ctrl+Enter to run single lines in RStudio
  • Press Tab for auto-completion of functions
  • Use %>% pipe operator for readable code
  • Save your workspace with .RData files
Debugging & Troubleshooting
  • Use View() to inspect data frames
  • Check column names with names()
  • Use summary() for quick data overview
  • Check for missing values with is.na()
Clinical Programming Best Practices
  • Always backup raw data before processing
  • Use consistent variable naming (e.g., USUBJID)
  • Validate CDISC compliance with admiral
  • Document derivation logic for QC review
